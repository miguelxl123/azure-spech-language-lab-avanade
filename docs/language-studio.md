# Azure Language Studio

O **Language Studio** permite analisar texto, entender intenções e extrair informações importantes:

- Análise de sentimento
- Classificação de texto
- Resposta a perguntas
- Compreensão de linguagem natural (NLU)

## Passos para utilização

1. Criar um recurso de **Language** no portal Azure.
2. Obter **chave de API** e **endpoint**.
3. Configurar o ambiente Python:
   ```bash
   pip install azure-ai-language-textanalytics
