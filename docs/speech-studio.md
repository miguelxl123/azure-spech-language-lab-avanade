# Azure Speech Studio

O **Speech Studio** permite criar soluções baseadas em fala, como:

- Transcrição de áudio em texto
- Reconhecimento de comandos de voz
- Conversão de texto em fala (Text-to-Speech)

## Passos para utilização

1. Criar um recurso de Speech no portal Azure.
2. Obter a **chave de API** e o **endpoint**.
3. Configurar o ambiente Python:
   ```bash
   pip install azure-cognitiveservices-speech
