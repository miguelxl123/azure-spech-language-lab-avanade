# Laboratório Azure Speech e Language Studio


![Azure](https://img.shields.io/badge/Azure-Microsoft-blue)
![Python](https://img.shields.io/badge/Python-3.10-blue)


Este repositório documenta a prática realizada no laboratório da DIO, explorando os serviços **Azure Speech Studio** e **Language Studio**. O objetivo é aplicar conceitos de análise de fala e linguagem natural, criando soluções de IA para voz e texto.

## Objetivos

- Praticar o uso do **Azure Speech Studio** para análise de áudio e transcrição.
- Praticar o uso do **Azure Language Studio** para análise de texto e compreensão de linguagem natural.
- Documentar processos técnicos de forma clara e estruturada.
- Compartilhar experiências e aprendizados no GitHub.

## Estrutura do Repositório

- `docs/` → Notas técnicas, insights e tutoriais sobre cada ferramenta.
- `examples/` → Exemplos práticos em Python para Speech e Language Studio.
- `.gitignore` → Arquivos a serem ignorados pelo Git.
- `LICENSE` → Licença do projeto.
